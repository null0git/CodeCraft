Distributed Mode — How It Works
  1. Keyspace built from charset + length rules
  2. Passwords encoded as numeric indexes
  3. Total combinations calculated
  4. Work divided among all idle workers
  5. Remainder distributed evenly
  6. Workers run in parallel, never overlapping
  7. Progress tracked per worker
  8. Password found → all workers stop immediately
  9. Worker crash → range reassigned automatically
